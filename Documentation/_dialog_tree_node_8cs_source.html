DialogTreeNode.IsAvaiable
virtual bool IsAvaiable()
Returns if it is possible to reach this node. The node can be BLOCKED by several reasons, such as the player not having the required PreConditions. In a DialogTreeNode, every node is reachable, but in inherited classes such as ConditionalTreeNode this method will check if the players has all the preconditions specified in a list.
